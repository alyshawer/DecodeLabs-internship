# DecodeLabs Internship Project 1

## Project Overview
This repository contains a rule-based chatbot built in Python for the DecodeLabs Industrial Training Kit 2026.

## File
- `Task-1.py` - A simple console chatbot that responds to predefined questions.

## How to run
1. Open a terminal in this folder.
2. Run:
   ```powershell
   python Task-1.py
   ```
3. Type a valid question like `hello`, `help`, or `what is ai`.
4. Type `exit` to quit.

## Notes
- The chatbot uses a dictionary lookup to return responses.
- Add more entries to the `responses` dictionary to expand its capabilities.
